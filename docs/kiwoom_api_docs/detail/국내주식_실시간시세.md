## 주문체결

**API ID**: `00`

### 기본 정보

- Method: `POST`
- URL: `/api/dostk/websocket`
- 모의투자: 지원
- WebSocket: 지원

### Request Parameters

```python
params = {
    "trnm": "",  # Required - 서비스명
    "grp_no": "",  # Required - 그룹번호
    "refresh": "",  # Required - 기존등록유지여부
    "item": "",  # Optional - 실시간 등록 요소
    "type": "",  # Required - 실시간 항목
    "return_code": "",  # Optional - 결과코드
    "return_msg": "",  # Optional - 결과메시지
    "trnm": "",  # Optional - 서비스명
    "type": "",  # Optional - 실시간항목
    "name": "",  # Optional - 실시간 항목명
    "item": "",  # Optional - 실시간 등록 요소
}
```

### Request Example

```json
{"trnm": "REG","grp_no": "1","refresh": "1","data": [{"item": [""],"type": ["00"]}
```
