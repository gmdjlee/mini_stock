## 공매도추이요청

**API ID**: `ka10014`

### 기본 정보

- Method: `POST`
- URL: `/api/dostk/shsa`
- 모의투자: 지원

### Request Parameters

```python
params = {
    "stk_cd": "",  # Required - 종목코드
    "tm_tp": "",  # Optional - 시간구분
    "strt_dt": "",  # Required - 시작일자
    "end_dt": "",  # Required - 종료일자
    "dt": "",  # Optional - 일자
    "close_pric": "",  # Optional - 종가
    "pred_pre_sig": "",  # Optional - 전일대비기호
    "pred_pre": "",  # Optional - 전일대비
    "flu_rt": "",  # Optional - 등락율
    "trde_qty": "",  # Optional - 거래량
    "shrts_qty": "",  # Optional - 공매도량
    "ovr_shrts_qty": "",  # Optional - 누적공매도량
    "trde_wght": "",  # Optional - 매매비중
    "shrts_trde_prica": "",  # Optional - 공매도거래대금
    "shrts_avg_pric": "",  # Optional - 공매도평균가
}
```

### Request Example

```json
{"stk_cd": "005930","tm_tp": "1","strt_dt": "20250501","end_dt": "20250519"}
```
