## 테마그룹별요청

**API ID**: `ka90001`

### 기본 정보

- Method: `POST`
- URL: `/api/dostk/thme`
- 모의투자: 지원

### Request Parameters

```python
params = {
    "qry_tp": "",  # Required - 검색구분
    "stk_cd": "",  # Optional - 종목코드
    "date_tp": "",  # Required - 날짜구분
    "thema_nm": "",  # Optional - 테마명
    "flu_pl_amt_tp": "",  # Required - 등락수익구분
    "stex_tp": "",  # Required - 거래소구분
    "thema_grp_cd": "",  # Optional - 테마그룹코드
    "thema_nm": "",  # Optional - 테마명
    "stk_num": "",  # Optional - 종목수
    "flu_sig": "",  # Optional - 등락기호
    "flu_rt": "",  # Optional - 등락율
    "rising_stk_num": "",  # Optional - 상승종목수
    "fall_stk_num": "",  # Optional - 하락종목수
    "dt_prft_rt": "",  # Optional - 기간수익률
    "main_stk": "",  # Optional - 주요종목
}
```

### Request Example

```json
{"qry_tp": "0","stk_cd": "","date_tp": "10","thema_nm": "","flu_pl_amt_tp": "1","stex_tp": "1"}
```

## 테마구성종목요청

**API ID**: `ka90002`

### 기본 정보

- Method: `POST`
- URL: `/api/dostk/thme`
- 모의투자: 지원

### Request Parameters

```python
params = {
    "date_tp": "",  # Optional - 날짜구분
    "thema_grp_cd": "",  # Required - 테마그룹코드
    "stex_tp": "",  # Required - 거래소구분
    "flu_rt": "",  # Optional - 등락률
    "dt_prft_rt": "",  # Optional - 기간수익률
    "stk_cd": "",  # Optional - 종목코드
    "stk_nm": "",  # Optional - 종목명
    "cur_prc": "",  # Optional - 현재가
    "flu_sig": "",  # Optional - 등락기호
    "pred_pre": "",  # Optional - 전일대비
    "flu_rt": "",  # Optional - 등락율
    "acc_trde_qty": "",  # Optional - 누적거래량
    "sel_bid": "",  # Optional - 매도호가
    "sel_req": "",  # Optional - 매도잔량
    "buy_bid": "",  # Optional - 매수호가
    "buy_req": "",  # Optional - 매수잔량
    "dt_prft_rt_n": "",  # Optional - 기간수익률n
}
```

### Request Example

```json
{"date_tp": "2","thema_grp_cd": "100","stex_tp": "1"}
```
