## 종목별투자자기관별차트요청

**API ID**: `ka10060`

### 기본 정보

- Method: `POST`
- URL: `/api/dostk/chart`
- 모의투자: 지원

### Request Parameters

```python
params = {
    "dt": "",  # Required - 일자
    "stk_cd": "",  # Required - 종목코드
    "amt_qty_tp": "",  # Required - 금액수량구분
    "trde_tp": "",  # Required - 매매구분
    "unit_tp": "",  # Required - 단위구분
    "dt": "",  # Optional - 일자
    "cur_prc": "",  # Optional - 현재가
    "pred_pre": "",  # Optional - 전일대비
    "acc_trde_prica": "",  # Optional - 누적거래대금
    "ind_invsr": "",  # Optional - 개인투자자
    "frgnr_invsr": "",  # Optional - 외국인투자자
    "orgn": "",  # Optional - 기관계
    "fnnc_invt": "",  # Optional - 금융투자
    "insrnc": "",  # Optional - 보험
    "invtrt": "",  # Optional - 투신
    "etc_fnnc": "",  # Optional - 기타금융
    "bank": "",  # Optional - 은행
    "penfnd_etc": "",  # Optional - 연기금등
    "samo_fund": "",  # Optional - 사모펀드
    "natn": "",  # Optional - 국가
    "etc_corp": "",  # Optional - 기타법인
    "natfor": "",  # Optional - 내외국인
}
```

### Request Example

```json
{"dt": "20241107","stk_cd": "005930","amt_qty_tp": "1","trde_tp": "0","unit_tp": "1000"}
```

### 기본 정보

- Method: `POST`
- URL: `/api/dostk/chart`
- 모의투자: 지원

### Request Parameters

```python
params = {
    "stk_cd": "",  # Required - 종목코드
    "base_dt": "",  # Required - 기준일자
    "upd_stkpc_tp": "",  # Required - 수정주가구분
    "cur_prc": "",  # Optional - 현재가
    "acc_trde_qty": "",  # Optional - 누적 거래량
    "acc_trde_prica": "",  # Optional - 누적 거래대금
    "open_pric": "",  # Optional - 시가
    "high_pric": "",  # Optional - 고가
    "low_pric": "",  # Optional - 저가
    "dt": "",  # Optional - 일자
    "pred_pre_sig": "",  # Optional - 전일대비기호
}
```

### Request Example

```json
{"stk_cd": "M04020000","base_dt": "20250826","upd_stkpc_tp": "1"}
```
