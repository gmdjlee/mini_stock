## 대차거래추이요청

**API ID**: `ka10068`

### 기본 정보

- Method: `POST`
- URL: `/api/dostk/slb`
- 모의투자: 지원

### Request Parameters

```python
params = {
    "strt_dt": "",  # Optional - 시작일자
    "end_dt": "",  # Optional - 종료일자
    "all_tp": "",  # Required - 전체구분
    "dt": "",  # Optional - 일자
    "dbrt_trde_cntrcnt": "",  # Optional - 대차거래체결주수
    "dbrt_trde_rpy": "",  # Optional - 대차거래상환주수
    "rmnd": "",  # Optional - 잔고주수
    "dbrt_trde_irds": "",  # Optional - 대차거래증감
    "remn_amt": "",  # Optional - 잔고금액
}
```

### Request Example

```json
{"strt_dt": "20250401",```Request Example```"end_dt": "20250430","all_tp": "1"}
```
